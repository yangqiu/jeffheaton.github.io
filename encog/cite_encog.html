Citing Encog

If you would like to cite Encog, please cite the paper Encog: Library of Interchangeable
Machine Learning Models for Java and C# that appeared in the June 2015 update of the
Journal of Machine Learning Research (JMLR).

For your convenience, several citation formats are provided below:

APA 6

Heaton, J. (2015). Encog: Library of Interchangeable Machine Learning Models for Java and C#. Journal of Machine Learning Research, 16, 1243-1247.

BibTex

Download Bibtex: heaton15a.bib

EndNote

Download EndNote: heaton15a.enw

MLA

Heaton, Jeff. “Encog: Library of Interchangeable Machine Learning Models for Java and C#.” Journal of Machine Learning Research 16 (2015): 1243-47. Print.
